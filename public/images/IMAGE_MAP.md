# HeLiPet Open Top Pro — Image Map

Đặt ảnh vào thư mục `public/images/` với tên tương ứng.
Ảnh nên ở định dạng `.webp` để tối ưu hiệu năng.

## Danh sách ảnh cần chuẩn bị

| File | Mô tả | Kích thước khuyến nghị |
|------|--------|----------------------|
| `1.webp` | **Hero** — Ảnh sản phẩm chính diện, nền trắng/trong suốt | 1200 x 900 px |
| `2.webp` | **Solution** — Ảnh sản phẩm góc 3/4 | 1000 x 750 px |
| `3.webp` | **Gallery 1** — Ngăn chứa chất thải | 800 x 600 px |
| `4.webp` | **Gallery 2** — Bảng điều khiển / nút bấm | 800 x 600 px |
| `5.webp` | **Gallery 3** — Mèo đang sử dụng | 800 x 600 px |
| `6.webp` | **Gallery 4** — Giao diện App trên điện thoại | 800 x 600 px |
| `avatar-1.webp` | Review — Avatar khách hàng 1 | 200 x 200 px |
| `avatar-2.webp` | Review — Avatar khách hàng 2 | 200 x 200 px |
| `avatar-3.webp` | Review — Avatar khách hàng 3 | 200 x 200 px |
| `avatar-4.webp` | Review — Avatar khách hàng 4 | 200 x 200 px |
| `avatar-5.webp` | Review — Avatar khách hàng 5 | 200 x 200 px |
| `og-image.webp` | OpenGraph / Social share image | 1200 x 630 px |

## Lưu ý
- Ảnh sản phẩm nên chụp trên nền trắng hoặc nền xám nhạt
- Ảnh avatar có thể là ảnh minh họa
- Sau khi đặt ảnh, không cần thay đổi code — website sẽ tự hiển thị
