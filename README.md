## 🚀 Cách chạy dự án (Getting Started)

Đầu tiên, cài đặt các thư viện cần thiết và chạy server phát triển (development server):

```bash
npm install
# sau đó chạy
npm run dev
```

Hoặc nếu bạn dùng yarn/pnpm:

```bash
yarn dev
# hoặc
pnpm dev
# hoặc
bun dev
```

Mở trình duyệt và truy cập [http://localhost:3000](http://localhost:3000) để xem kết quả.
